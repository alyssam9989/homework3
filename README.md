# Homework Week 3
## Alyssa's Baking Cost Calculator

### Description

This program calculates the total cost of a recipe based on what dessert you are making, as well as the quantity needed.  

### Inputs

1. What are you baking?
2. How much does it cost to make one?
3. How many do you need to make?
4. What is your budget? 

### Outputs

1. Summary of recipe.
2. Total Cost of recipe.
3. Indication of budget parameters.
4. Exit
